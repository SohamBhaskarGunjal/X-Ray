# X-Ray Hand Tracker
X-RAY HAND TRACKER . GESTURE INTERACTIONS & CONTROL Show your hands to the webcam. Splay fingers to track, or wait for the auto-breathing test box.
A real-time, browser-based hand-tracking application that overlays a stylized, skeletal X-ray visual effect onto detected hand movements using computer vision.

## 🚀 Live Demo

Check out the live application here: [https://x-ray-hand-tracker-beige.vercel.app/](https://x-ray-hand-tracker-beige.vercel.app/)

---

## ✨ Features

* **Real-time Tracking:** High-performance hand detection directly through your webcam.
* **X-Ray Visuals:** A creative overlay that tracks and highlights skeletal hand structures.
* **Zero Installation:** Fully browser-based application—no setup required for users.
* **Responsive Design:** Optimized to work across various screen sizes.

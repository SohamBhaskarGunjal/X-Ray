# 📖 About The Project

**X-Ray Hand Tracker** is an interactive computer vision web application that brings a futuristic X-Ray experience directly to your browser.

Using your webcam, the application detects your hands in real time and renders a glowing skeletal overlay that accurately follows every finger movement.

Unlike traditional desktop applications, this project runs entirely inside the browser without requiring any installation, plugins, or additional software.

The project demonstrates how modern web technologies can deliver high-performance computer vision experiences while maintaining smooth animations and responsive interactions.

Whether you're exploring computer vision, experimenting with gesture recognition, or simply enjoying futuristic UI effects, X-Ray Hand Tracker showcases what's possible with browser-based technologies.

# ⚙️ How It Works

1. Open the application.
2. Allow webcam permission.
3. The browser starts capturing live video.
4. Your hands are detected in real time.
5. Hand landmarks are generated instantly.
6. A glowing X-Ray skeleton is rendered over each detected hand.
7. Move your hands naturally and watch the overlay respond in real time.
